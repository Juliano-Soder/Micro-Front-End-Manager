Configuração Manual - Nexus

Siga os passos abaixo para configurar manualmente seu acesso ao Nexus

  1. 1
    Navegar até a pasta do projeto
    Abra um terminal e navegue até a pasta de um dos seus projetos (navbar, marketplace, etc)
    cd D:\workdir\front-end\mp-pas-navbar
    ℹ️ Substitua o caminho pelo seu projeto (mp-pas-marketplace, mp-pamp, etc)
  2. 2
    Configurar o registry do Nexus
    Adicione o registry do Nexus à configuração do npm
    npm config set registry http://nexus.viavarejo.com.br/repository/npm-marketplace/
  3. 3
    Inserir seu email
    Configure seu email corporativo
    npm config set email "seu-email@empresa.com.br"
  4. 4
    Inserir sua senha
    Configure sua senha do Nexus
    npm config set _password "sua-senha"
  5. 5
    Verificar se está logado no Nexus
    Confirme que a autenticação funcionou corretamente
    npm ping
    ✅ Se retornar "Packages" você está logado com sucesso!
⚠️ Importante: Substitua os valores de exemplo pelas suas credenciais reais (seu email corporativo e sua senha do Nexus).
